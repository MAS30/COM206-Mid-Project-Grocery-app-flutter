# Smart Grocery Cart Mobile Application (Flutter)

# Project Overview
This project is a mobile-based grocery cart application developed using Flutter and Dart. It provides an interactive system where users can browse products by category, adjust quantities, and manage a shopping cart dynamically.

The application demonstrates how mobile interfaces can be combined with logical data handling to solve real-world problems related to grocery shopping.

# Problem Statement
Managing grocery shopping efficiently can be challenging when users need to track items, quantities, and total cost simultaneously. Traditional methods lack structure and can lead to confusion or inefficient planning.

This project addresses the problem by providing a simple and organized mobile application where users can filter products, add them to a cart, and manage their selections dynamically.

# Objectives
- To develop a mobile application using Flutter  
- To implement a dynamic grocery cart system  
- To demonstrate event-driven programming  
- To manage data using lists and state updates  
- To provide a clean and user-friendly interface  

# Features
- Category-based product filtering (Fruits, Vegetables, Dairy)  
- Add to cart functionality  
- Update item quantity in cart  
- Remove items from cart  
- Dynamic total calculation  
- Real-time UI updates using setState()  
- User feedback using SnackBar messages  

# System Logic
The application follows an event-driven approach:

- User actions trigger functions (Add, Remove, Update)  
- Products are filtered based on selected category  
- Cart items are stored in a list data structure  
- Existing items are updated instead of duplicated  
- UI updates dynamically using Flutter state management  

# Workflow
1. User opens the application  
2. Selects a category  
3. Views filtered products  
4. Adjusts item quantity  
5. Clicks "Add to Cart"  
6. Item is stored or updated in cart  
7. Cart and total are updated dynamically  
8. User can modify or remove items  

# Technologies Used

- Dart (Programming Language)  
- Material UI Components
